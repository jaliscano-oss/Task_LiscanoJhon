Proceso de Aprendizaje en Desarrollo Web

Unidad 1 — Fundamentos de HTML

Se aprendió la estructura básica de un documento HTML, etiquetas semánticas, listas, tablas, formularios, multimedia (audio, video, iframe) y rutas relativas entre páginas. Se practicó la organización de proyectos en carpetas.

Unidad 2 — Fundamentos de CSS

Se aplicaron estilos visuales usando CSS externo. Se practicaron selectores de etiqueta, clase e ID. Se comprendió el Box Model (margen, borde, relleno, contenido), Flexbox para maquetación y media queries para diseño responsivo.

Proyecto Final — PracticaCSSFinal

Desarrollo completo de un sitio web multi-página con diseño visual coherente, navegación entre páginas, CSS modular por archivo y adaptación responsiva para dispositivos móviles y escritorio.

Reflexión Personal

Durante este período aprendí que el desarrollo web es mucho más que escribir código: es entender al usuario y pensar en cómo este interactuará con la página desde cualquier dispositivo.

El mayor desafío fue comprender el Box Model y cómo los márgenes, bordes y rellenos afectan el diseño. Con práctica constante, logré aplicar estos conceptos de manera efectiva en el proyecto final.

"El diseño web no se trata solo de hacer que las cosas se vean bien, sino de hacer que funcionen bien para todos."
